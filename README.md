
# DevKit 🚀

DevKit is a lightweight **developer toolkit web application** that provides commonly used development utilities in one place. It helps developers quickly perform tasks like formatting JSON, encoding/decoding Base64, decoding JWT tokens, and converting timestamps directly in the browser.

The project is built using **HTML, CSS, and Vanilla JavaScript**, making it fast, simple, and easy to deploy anywhere. DevKit also supports **Progressive Web App (PWA)** features so it can be installed and used offline.

---

## ✨ Features

- JSON Formatter & Validator
- Base64 Encode / Decode
- JWT Decoder
- Timestamp Converter
- Clean and modern UI
- Fast client‑side processing
- Installable as a **Progressive Web App (PWA)**
- Offline support using **Service Worker**

---

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/yourusername/devkit.git
```

### 2. Open the project

Simply open:

```
index.html
```

in your browser.

Or run a local server for better PWA support.

---

## 📱 PWA Installation

If hosted on HTTPS or localhost:

1. Open the site in Chrome or Edge
2. Click **Install App**
3. DevKit will be installed like a native application

You can then launch it directly from your device.
