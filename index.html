Getting Started

DevKit is a zero-install, browser-based collection of 11 developer utilities. No sign-up, no backend, no data sent anywhere.

1

Open DevKit

Open devtools.html in any modern browser. Works offline after first load.

2

Search or Scroll

Use the search bar in the header to instantly filter tools by name or keyword.

3

Paste, Click, Copy

Each tool follows the same pattern: paste input → click action → copy output.

💡 Privacy100% client-side. Your data never leaves your machine — safe for tokens, API keys, and private configs.

📲
PWA / Install as App

DevKit is a Progressive Web App. Install it to your device for instant offline access from any homescreen or desktop.

1

Look for the Install Button

When your browser supports PWA installation, an "Install App" button appears in the header and a toast notification at the bottom.

2

Click Install & Confirm

Click install and confirm the browser prompt. DevKit appears as a standalone app on your device.

💡 OfflineOnce installed, all 11 tools work with zero internet connection.

🔍
Search Bar

The search bar in the header filters all 11 tool cards in real-time. Type any keyword: tool name, technology, or use-case.

Examples: "hash" - shows Hash Generator "jwt" - shows JWT Decoder "base" - shows Base64 + Number Base "encode" - shows Base64 + URL Encoder "markdown" - shows Markdown Previewer
💡 TipClick the X button or press Backspace to clear the search and show all tools again.

{ }
JSON Formatter

Parse, format, minify, and validate JSON data. Catches syntax errors with precise messages.

⚠ JSON keys must be double-quoted strings. Single quotes will fail validation.

64
Base64 Encode / Decode

Convert plain text to Base64 and back. UTF-8 safe — handles Unicode, emoji, and special characters correctly.

🔗
URL Encode / Decode

Percent-encode special characters in URLs for safe transmission, or decode them back to readable form.

#
Hash Generator

Generate SHA-1, SHA-256, and SHA-512 cryptographic hashes using the browser's native WebCrypto API. Results update automatically as you type.

Input: "hello" SHA-1: aaf4c61ddcc5e8a2dabede0f3b482cd9aea9434d SHA-256: 2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c... SHA-512: 9b71d224bd62f3785d96d46ad3ea3d73319bfbc2...
💡 Use CaseVerify file integrity, generate checksums, or safely compare strings without exposing originals.
⚠ SHA-1 is cryptographically weak. Use SHA-256 or SHA-512 for security-critical applications.

🔑
JWT Decoder

Decode and inspect JSON Web Tokens. Header, payload, and signature displayed in color-coded panels. Token is decoded locally — never sent anywhere.

⚠ This tool only DECODES JWTs — it does NOT verify signatures. Do not treat decoded data as trusted without server-side verification.

M↓
Markdown Previewer

Write Markdown in the editor and see it rendered live. Supports headings, bold, italic, lists, code blocks, blockquotes, tables, and links.

# Heading · **bold** · _italic_ · `code` - list · > blockquote · [link](url) ```js code block ```

01
Number Base Converter

Convert between Binary (2), Octal (8), Decimal (10), and Hexadecimal (16). Select your input base, type a number, and all four formats update instantly.

Decimal 255 → Binary: 11111111 | Octal: 377 | Hex: FF

Aa
Text Analyzer

Real-time word count, character count, line count, and sentence count as you type or paste. Useful for SEO, copywriting constraints, and content writing.

.*
Regex Tester

Test JavaScript regular expressions against any string. Reports match count and exact positions. Global flag always applied.

⚠ Patterns with catastrophic backtracking may freeze the browser tab. Test complex patterns on small inputs first.

⏱
Timestamp Converter

Convert Unix timestamps to ISO, UTC, and local formats. 10-digit = seconds, 13-digit = milliseconds. Hit Now for the current time instantly.

🎨
Color Converter

Convert between HEX, RGB, HSL, and CSS custom property formats. Use the native picker or type a hex value directly. Click any result card to copy.

❓
FAQ

Is my data sent to any server?

No. Every tool runs 100% in your browser using native APIs. Nothing is logged or transmitted. Safe for passwords, API keys, JWTs, and sensitive configs.

How do I install it as an app?

Look for the "Install App" button in the header or the toast at the bottom of the screen. Click it and confirm the browser prompt. The app then works fully offline.

Can I add my own tools?

Yes — open the HTML file, duplicate a .tool-card block, add a data-tags attribute, and write your JS logic. Contributions on GitHub are also welcome.

Which browsers are supported?

Chrome 90+, Firefox 88+, Safari 14+, Edge 90+. The Hash tool requires WebCrypto, available in all modern browsers on https:// or file:// contexts.
